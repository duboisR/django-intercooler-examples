Django Intercooler Examples

Intercooler is a client-side library that makes it easy to define AJAX requests via HTML attributes, and expects HTML response that gets inserted into the page. This library seems like the best candidate for boosting a Django project without javascript.

Below are a series of examples demonstrating common UI/UX problems being solved with intercooler. All examples are inspired by the Intercooler documentation.

Examples

Title Description
Click To Edit A simple demo of inline editing of data within a page.
Infinite Scroll Loading more elements when the bottom of a page is scrolled into view.
Click To Load Similar to the infinite scroll demo, but requires that a user explicitly click on a button to load more items.
Bulk Update UI A demo of a table-driven UI, where multiple rows are updated at once via checkbox selection.
